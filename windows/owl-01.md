## System Under Test
Windows Calculator

## Preconditions
- Calculator should be in Standard mode with the display cleared

## Test Intent
Multiply 21 by 2

## Expected Result
The calculator display shows exactly "42"

## Test Steps

Step 1 — Open Calculator
- Launch Calculator via Start Menu or `calc.exe`

Step 2 — Clear the display
- Press 'Escape' to ensure the display is reset to 0

Step 3 — Enter '2' then '1'
- Press '2', then '1' so the display reads "21"

Step 4 — Press multiply
- Press '*' to enter the multiplication operator

Step 5 — Enter '2'
- Press '2' so the display reads "2"

Step 6 — Press equals
- Press '=' to compute the result

Step 7 — Verify result
- Confirm the calculator display shows exactly "42"
